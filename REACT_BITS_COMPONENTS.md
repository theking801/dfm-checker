## 📦 Comment utiliser

Les composants sont dans `frontend/node_modules/reactbits-mcp-server/src/content/`.

Pour récupérer un composant :
```bash
# Lister les composants
find frontend/node_modules/reactbits-mcp-server/src/content -name "*.jsx" | sort

# Voir le code source d'un composant
cat frontend/node_modules/reactbits-mcp-server/src/content/<Category>/<Component>/<Component>.jsx

# Voir le CSS associé
cat frontend/node_modules/reactbits-mcp-server/src/content/<Category>/<Component>/<Component>.css
```
